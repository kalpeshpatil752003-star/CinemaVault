## ğŸ› ï¸ Customization

### Adding New Movies

1. Open `data.ts`
2. Add a new movie object to the `movies` array:
```typescript
{
  id: "13",
  title: "Your Movie Title",
  year: 2024,
  rating: 4.5,
  reviewCount: 1000,
  genres: ["Action", "Drama"],
  poster: "https://your-image-url.jpg",
  synopsis: "Your movie description...",
  director: "Director Name",
  cast: ["Actor 1", "Actor 2"],
  runtime: 120,
  trailer: "https://www.youtube.com/embed/VIDEO_ID",
  reviews: [
    {
      id: "r37",
      author: "Reviewer Name",
      rating: 5,
      comment: "Great movie!",
      date: "2024-01-01"
    }
  ]
}
```
3. Recompile TypeScript:
```bash
npx tsc --project tsconfig.cinema.json
```

### Modifying Styles

Edit `styles.css` to change:
- Colors (CSS variables at the top)
- Layout (grid columns, spacing)
- Typography (fonts, sizes)
- Animations (transitions, hover effects)
